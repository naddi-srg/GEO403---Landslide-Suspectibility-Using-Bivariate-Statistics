This project provides a reproducible workflow for landslide susceptibility analysis using Frequency Ratio (FR), the Statistical Index (WI), and a weighted model (WI + WF).The repository is structured so that the analysis can be reproduced with a small example dataset. All file paths in the script are defined relative to the project directory, so the workflow can be executed without modifying paths after downloading the folder.The main script is located in the scripts directory. It performs all steps of the analysis, including preprocessing of raster data, clipping to the viewshed, calculation of FR, WI and WF values, creation of susceptibility maps, classification based on percentile ranking, and validation using an 80/20 train-test split.The data_example folder contains a reduced dataset that allows the workflow to be tested. It includes classified raster factors, a viewshed shapefile, and landslide point data. The full dataset is not included due to file size limitations and is provided separately via cloud storage.To run the workflow, install the required Python packages listed in requirements.txt, navigate to the scripts directory, and execute the main script. All outputs are written to the results folder, which is created automatically.The workflow produces lookup tables for FR, WI and WF, factor-based raster layers, final susceptibility maps for both models, classified maps based on percentile thresholds, and validation results including success rate, prediction rate, and AUC values.This setup is intended to ensure that the entire analysis can be reproduced and understood without requiring access to the full original dataset.
